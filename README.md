## Run Locally

Clone the project

```bash
  git clone https://github.com/vasquezluis/levelup_proyect3_frontend
```

Go to the project directory

```bash
  cd levelup_proyect3_frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
